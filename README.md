# PaperSourcing
紙のデータを皆で手入力でデジタル化するプラットフォームです。

## 動作環境
* node.js v0.11.x(可能な限り新しいことが望ましい)
* PostgreSQL 9.1+
* Redis

## デプロイ方法
* npm install
* node app.js
